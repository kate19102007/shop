# 🛒 Online Store Console App

**Online Store** — це консольний застосунок на C++, що імітує роботу автоматизованої системи супермаркету. Програма дозволяє керувати кошиком, застосовувати промокоди, враховувати бонуси карти лояльності та генерувати фіскальні чеки.

## 🌟 Основні можливості

* 📦 **Керування кошиком:** Додавання, видалення товарів та зміна їх кількості в реальному часі.
* 💰 **Динамічний прайс-лист:** Автоматичне завантаження цін із зовнішнього файлу `products.txt`.
* 🎫 **Система знижок:** Підтримка промокодів (наприклад, `SALE10`, `SALE20`).
* 💳 **Програма лояльності:** Розумна система акцій, що автоматично нараховує безкоштовні товари або знижує ціну.
* 🧾 **Генерація чеків:** Вивід детального звіту про покупку та збереження історії замовлень у файл `orders.txt`.
* 🚚 **Симуляція доставки:** Анімований статус виконання замовлення.

---

## 🎁 Акції програми лояльності

Якщо у користувача є карта лояльності, система автоматично перевіряє кошик на відповідність таким умовам:

| Товар | Умова акції | Вигода |
| :--- | :--- | :--- |
| **Banana** | Купуй 2 | **+1 безкоштовно** |
| **Apple** | Купуй 3 | **+1 безкоштовно** |
| **Yogurt** | Якщо в чеку є 2+ одиниці молока | **Знижка -15%** |
| **Cookies** | Якщо купується разом із кавою | **Знижка -20%** |
| **Pizza** | При купівлі будь-якої піци | **Cola у подарунок** |
| **Rice/Pasta** | Якщо в чеку є 2+ одиниці курки | **Знижка -10%** |

## 🛠 Технічні деталі

* **Мова:** C++
* **Бібліотеки:** `STL` (vector, map, string), `fstream` (файлова система), `chrono/thread` (анімація затримок).
* **Зберігання даних:** * `products.txt` — база товарів та цін.
    * `orders.txt` — лог усіх оформлених замовлень.

## 🚀 Як запустити

1.  Переконайтеся, що у вас встановлено компилятор C++ (GCC, Clang або MSVC).
2.  Створіть файл `products.txt` у кореневій папці з таким вмістом:
    ```text
    milk 35.5
    banana 60.0
    apple 25.0
    coffee 120.0
    pizza 240.0
    ```
3.  Скомпілюйте та запустіть код:
    ```bash
    g++ main.cpp -o store
    ./store
    ```
